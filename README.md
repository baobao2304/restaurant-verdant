# Verdant — Fine Dining Restaurant

Landing page nhà hàng cao cấp, tone xanh lá đậm sang trọng với accent champagne.

## Tech
- HTML5 + CSS3 thuần (không framework)
- Vanilla JavaScript (sticky nav, mobile menu, reveal-on-scroll, reservation form demo)
- Google Fonts: Cormorant Garamond + Jost

## Sections
Hero · About · Menu · Chef · Gallery · Reservation · Footer

Mỗi section dùng một loại họa tiết nền khác nhau (geometric / dots / lines / glow / organic leaf SVG) để tạo chiều sâu mà không rối mắt.

## Chạy local
```bash
# mở trực tiếp
open index.html
# hoặc serve
python3 -m http.server 8080
```

## Deploy
Đẩy lên GitHub → bật GitHub Pages (branch `main`, folder `/root`) là có link live.
